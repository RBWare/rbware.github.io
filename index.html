US Constitution

The United States Constitution application was one of the first 2000 applications available on the Android platform. It arrived to the Android Market prior to the 1.5 launch and has undergone many revisions since that time. It uses the open source libraries ActionBarSherlock and HoloEverywhere to maintain compatibility with devices prior to Android 4.x (2.2+ is supported). The application was originally just a proof-of-concept for a college project but has been maintained with several updates since its original launch. The code is available for free to other developers who are interested in creating similar projects.

Glass App Manager

A lightweight application designed to allow the user to manage application information directly from Google Glass. As of version 2.0 of the Glass OS (a custom tailored version of Android), the only way to manage pre-installed applications was to connect the device via adb and manually use commands like adb uninstall package_name. The Glass App Manager implements a UI for this directly on the Glass hardware. Using a simple horizontal list, an application can be selected and with just a few taps several actions can be taken. The application allows you to "Force Stop", "Clear Data", "Clear Cache", and "Uninstall" without the need to ever connect to a computer. The application can be triggered by selection, or by saying, "Okay Glass, Uninstall an App."

Source Code | License: Apache 2.0

Glass Photo Gallery

Photo Gallery application for Google Glass. This application solves the issue that as soon as a photo or video is taken, the user is only given a very brief moment to review what was captured before it is stored locally and no longer accessible. For some reason, as part of Google's decision with Glass, a photo or video can only be reviewed one time locally before it needs to be handed off to a computer or cloud service. With this application, the user can go back and review previously captured photo and video at their leisure. It also allows for a limited set of options like "Deleting", directly from the device. Without this application, users would be forced to attach glass via MicroUSB to their computer, and manually review and delete the content. This behavior was especially frustrating as the local storage started to fill up.

Source Code | License: Apache 2.0

Fire TV Reboot

Lightweight application to reboot the Fire TV. Root access required. By default, there is no way, short of physically power-cycling the Fire TV for users to reboot the Amazon Fire TV. This application lets you "start" an application that reboots the FireTV programmatically.

Android Image Script

A small script designed to help solve a disconnect between Designers and Developers. When designers create assets for Android applications, it is generally done in an svg format. However, unlike iOS, Android does not natively handle SVG images. This script takes an SVG image as input and outputs android-compatible PNG images for densities LDPI through XXHDPI which saves time for designers as well devs.
